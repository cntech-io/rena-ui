# @cntech/rena-ui

Custom components for fast development.

## Install

```bash
npm install @cntech/rena-ui
```

```bash
yarn add @cntech/rena-ui
```

## Peer dependencies

```bash
npm install react-native-linear-gradient
```

```bash
yarn add react-native-linear-gradient
```

## Components

- Button
- ButtonGroup
- Container
- Flex
- Header
- Icon
- IconButton
- Input
- Spacer
